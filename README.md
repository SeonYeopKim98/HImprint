# HIimprint

> **HMM-based Imprinting Control Region Detection Tool for Long-Read Sequencing**

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Linux%20%7C%20macOS-lightgrey)]()

**HIimprint** is a high-performance bioinformatics tool designed to detect **Imprinted Control Regions (ICRs)** and **Differentially Methylated Regions (DMRs)** from phased long-read sequencing data (e.g., PacBio HiFi). By employing a **3-state Hidden Markov Model (HMM)** and **Run-Length Encoding (RLE)** filtering, HIimprint robustly identifies parent-of-origin-specific methylation patterns while minimizing stochastic noise.

---

## Key Features

- 3-State HMM — Models Hap1-methylated, Hap2-methylated, and unimprinted states per CpG site
- RLE Filtering — Suppresses short spurious calls and retains only contiguous imprinted segments
- Chromosome-level Parallelism — Dask-based multi-threading for efficient genome-wide analysis
- Phased Input Support — Directly compatible with pb-cpg-tools haplotype BED output
- Built-in QC — Per-site coverage filtering to exclude low-quality CpGs before HMM decoding

## Usage

HIimprint supports two input modes:

### Mode 1 — From pb-cpg-tools BED Files

Provide two phased methylation BED files generated by [pb-cpg-tools](https://github.com/PacificBiosciences/pb-CpG-tools):

```bash
python hiimprint.py \
  --hap1 hap1.bed \                      # Haplotype 1 BED (pb-cpg-tools output)
  --hap2 hap2.bed \                      # Haplotype 2 BED (pb-cpg-tools output)
  -c 0.8 \                               # Methylation difference cutoff (default: 0.8)
  -l 10 \                                # Minimum segment length after decoding (default: 10)
  --pe 0.9 \                             # Major emission probability (default: 0.9)
  --pt 0.9 \                             # Major transition probability (default: 0.9)
  -t 16 \                                # Dask workers (default: 24)
  -o sample.fc_c0.8_l10.res.txt          # Output path (optional)
```

### Mode 2 — From Pre-calculated Methylation Difference File

If you already have a pre-calculated methylation difference file (`chrom`, `start`, `end`, `meth_diff`):

```bash
python hiimprint.py \
  -i methylation_diff.tsv \              # Pre-calculated methylation difference file (TSV)
  -c 0.8 \                               # Methylation difference cutoff (default: 0.8)
  -l 10 \                                # Minimum segment length after decoding (default: 10)
  --pe 0.9 \                             # Major emission probability (default: 0.9)
  --pt 0.9 \                             # Major transition probability (default: 0.9)
  -t 16 \                                # Dask workers (default: 24)
  -o sample.fc_c0.8_l10.res.txt          # Output path (optional)
```

## Parameters

| Argument | Short | Default | Description |
|---|---|---|---|
| `--input` | `-i` | — | Pre-calculated methylation difference file (TSV) |
| `--hap1` | — | — | Haplotype 1 BED file |
| `--hap2` | — | — | Haplotype 2 BED file |
| `--output` | `-o` | auto | Output file path |
| `--cutoff` | `-c` | `0.8` | Methylation difference threshold to call imprinting |
| `--length` | `-l` | `10` | Minimum number of consecutive CpGs to define a region |
| `--threads` | `-t` | `16` | Number of parallel threads (Dask) |
| `--min_cov` | — | `1` | Minimum coverage per CpG for both haplotypes |
| `--pe` | — | `0.9` | HMM emission probability (major state) |
| `--pt` | — | `0.9` | HMM transition probability (major state) |

## Input Format

### Mode 1 — Haplotype BED file (pb-cpg-tools output)

Tab-separated; HIimprint uses columns 1, 2, 6, and 9:

```
chr1    10468    10469    .    .    25    .    .    85.0
chr1    10470    10471    .    .    30    .    .    90.0
...
```

| Column | Description |
|---|---|
| 1 | Chromosome |
| 2 | Start position |
| 6 | Coverage |
| 9 | Methylation percentage (0–100) |

### Mode 2 — Pre-calculated Methylation Difference file

Tab-separated with a header line; methylation difference should be in the range of `-1.0` to `1.0`:

```
chrom    startp    endp    meth
chr1     10468     10469   0.85
chr1     10470     10471   -0.72
...
```

| Column | Description |
|---|---|
| chrom | Chromosome |
| startp | Start position |
| endp | End position |
| meth | Methylation difference (Hap1 − Hap2), range: −1.0 to 1.0 |

## Method Overview

```
Phased BED files (Hap1 / Hap2)
         │
         ▼
  Methylation Difference Calculation
  (Hap1 - Hap2) / 100 per CpG site
         │
         ▼
  Binary Classification per CpG
  diff ≥ cutoff  → state 1
  diff ≤ -cutoff → state 2
  otherwise      → state 0
         │
         ▼
  3-State HMM (per chromosome, in parallel)
  States: Hap1-methylated / Hap2-methylated / Unimprinted
         │
         ▼
  RLE Filtering (min_length threshold)
         │
         ▼
  ICR / DMR regions output
```
